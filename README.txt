KIT DE DÉMARRAGE LEAFLET

1. Ouvrez le fichier index.html dans un navigateur.
2. Le fichier main.js contient le code Leaflet que vous pouvez modifier.
3. Les données sont à placer dans le dossier /data en format GeoJSON.
4. Personnalisez le style avec style.css si besoin.

Ce kit fonctionne directement avec une connexion Internet grâce aux CDN.
